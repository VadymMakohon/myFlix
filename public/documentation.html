myFlix App

Short description

I am developing the backend for a web application focused on movies. This platform will offer users access to extensive details about various movies, including information about directors, release year, and genres. Additionally, users will have the ability to register, modify their profiles, and curate a collection of their favorite films.

Server

MOVIES: Managing movies API

GET
/movies Lists all the movies
GET
/movies/{Title} Lists all data about a movie by the title
GET
/movies/genres/{Genre} Lists all the movie titles with a specific genre
GET
/movies/directors/{Director Name} Lists data about a director

USERS: Managing users API

GET
/users Lists all the users
POST
/users Create a new user
PUT
/users/{UserName} Update user name using the UserName
DELETE
/users/{id} Remove user from the list
POST
/users/{UserName}/movies/{Title} Add a movie to the user's favorite movies list
DELETE
/users/{UserName}/movies/{Title} Delete a movie from the user's favorite movies list
